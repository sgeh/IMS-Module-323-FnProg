TypeScript-Vorlage

Siehe TypeScript-File Source unter /src/main.ts.

Die Kompilation des TS-Codes in JavaScript erfolgt auf der Konsole mittels npm run build im Root-Ordner (oberste Ebene, wo das package.json File liegt). Das kompilierte (übersetzte) JavaScript File ist im Anschluss unter scripts /scripts/main.js verfügbar.
